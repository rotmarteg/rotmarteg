## 🏗️ Projeto em Destaque: Smart Load 2.0
O **Smart Load 2.0** é o meu projeto principal, desenvolvido para ser o "braço direito" do conferente no pátio logístico.

*   **Problema Solucionado:** Mitigação da assimetria de informação e redução de avarias no manuseio de vidros.
*   **Inovação:** Renderização 3D em tempo real para visualização de racks e caixas, permitindo um sequenciamento de carga preditivo.
*   **Impacto:** Redução do *lead time* de carregamento e aumento da autoridade técnica operacional.
